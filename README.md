## Apollo GraphQL Setup

#### Prerequisites
- Install Node 10v
- Any Code Editor
- Nodemon (Optional)

#### Initial Steps
- git clone https://github.com/instamint/gql-service-be.git
- cd gql-service-be
- npm install
- Setup your Database credentials on utilles/Connection.js
- simply run node server.js
